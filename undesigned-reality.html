GRAPHIC

critical design poster:

undesigned reality

 

Year: 2015

Client: none

Designer: Jae Yong Lee, EMMA-SOFIE KUKKONEN
Parts of participation: graphic design
Participation: 100%

This poster designs were started with a critical perspective to the difference between designers’ intentions and the real use of design products in people’s context.

 

'Are people using the design objects in the context for which the designer has designed them for?'

 

When designing objects, the designer has an idea of the environment, in which the designed item will be used. The designer gives the shape to an object but is not able to effect to the circumstances in which the object is finally used. The usage context itself has an impact to the way how the object is seen and continues shaping the item in a way that is not predictable.
